# AI-Based Autonomous Drone Detection and Tracking System

## Overview

This project develops an intelligent surveillance system capable of detecting and tracking drones using computer vision and embedded systems.

The objective is to provide a low-cost prototype demonstrating autonomous aerial object detection and real-time tracking.

---

## Features

- Drone Detection
- Real-Time Object Tracking
- Pan-Tilt Camera Control
- ESP32 Integration
- Computer Vision
- Modular Design
